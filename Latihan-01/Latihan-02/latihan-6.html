Python

Python adalah bahasa pemrograman yang populer untuk pengembangan web, data science, dan AI.

JavaScript

JavaScript digunakan untuk membuat halaman web interaktif dan aplikasi berbasis web.

Java

Java adalah bahasa pemrograman yang banyak digunakan dalam aplikasi enterprise dan Android.

C#

C# adalah bahasa yang dikembangkan oleh Microsoft, sering digunakan dalam pengembangan aplikasi desktop dan game.

C

C adalah bahasa pemrograman tingkat rendah yang digunakan untuk pengembangan sistem dan aplikasi.

C++

C++ adalah bahasa yang lebih canggih dari C dan sering digunakan dalam pengembangan game dan aplikasi high-performance.

Go

Go adalah bahasa yang dikembangkan oleh Google, dirancang untuk performa tinggi dan efisiensi.

R

R adalah bahasa pemrograman yang banyak digunakan dalam analisis data dan statistik.

Swift

Swift adalah bahasa yang digunakan untuk mengembangkan aplikasi iOS dan macOS.

PHP

PHP adalah bahasa yang banyak digunakan dalam pengembangan web backend.
